# Useful loci lists
This directory contains loci lists that may be useful for clustering analysis in routine surveillance:

- _cgMLST_Salmonella_enterica_chewie-NS.txt_ - List of cgMLST loci of the [INNUENDO](https://doi.org/10.2903/sp.efsa.2018.EN-1498) wgMLST schema available at [chewie-NS for _Salmonella enterica_](https://chewbbaca.online/species/8)
  
- _cgMLST_Escherichia_coli_chewie-NS.txt_ - List of cgMLST loci of the [INNUENDO](https://doi.org/10.2903/sp.efsa.2018.EN-1498) wgMLST schema available at [chewie-NS for _Escherichia coli_](https://chewbbaca.online/species/5)
  
- _cgMLST_Campylobacter_jejuni_chewie-NS.txt_ - List of cgMLST loci of the [INNUENDO](https://doi.org/10.2903/sp.efsa.2018.EN-1498) wgMLST schema available at [chewie-NS for _Campylobacter jejuni_](https://chewbbaca.online/species/4)
  
- _cgMLST_Legionella_pneumophila_LIT.txt_ - List of cgMLST loci of the [LIT wgMLST schema](https://www.medrxiv.org/content/10.64898/2026.02.18.26346554v1) available at [Zenodo for _Legionella pneumophila_](https://doi.org/10.5281/zenodo.17871973)

- _cgMLST_Legionella_pneumophila_LIT_chewie-NS.txt_ - List of cgMLST loci of the [LIT wgMLST schema](https://www.medrxiv.org/content/10.64898/2026.02.18.26346554v1) available at [chewie-NS for _Legionella pneumophila_](https://chewbbaca.online/species/19)
